----------INSTITUIONAL WEBSITE/LANDING PAGES PROCESS OF CREATION-----------

1) Create the prototipe on Figma.

2) Donwload all images, logos and favicons from Unplash/Pexels/Freepik, 
edit it if necessary and remove their sizes on https://tinypng.com.

3) Clone the repository template from GitHub.

4) Configure colors, and fonts and provide the images.

5) Code the site.

6) Check the reponsivity and improve it. 

7) Check the SEO and improve it.

8) Check all importations, unused components, unused packages 
and deploy the application.

9) Check the website metrics (performance, seo and layout shifiting) and 
improve it if necessary.
