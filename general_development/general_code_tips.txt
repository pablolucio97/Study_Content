=================GENERAL CODE TIPS-===================

•props.children displays the content of the HTML 
element.

------------------------------------------------------

•Use globals.css to style your all application.

------------------------------------------------------

•Always use preventDefault() at working with forms 
in react.

------------------------------------------------------

•To merge two or more arrays in react, use the 
syntax: [...vector1, ...vector2]

------------------------------------------------------

•Use Javascript syntax instead of sql syntax to 
relation with databases.

------------------------------------------------------

•Create the entities and functionalities of your 
application thinking in the front end web and mobi-
le (if the will exists).

------------------------------------------------------

•Use the crypto.randomBytes() to generate hexade-
cimal characters.

------------------------------------------------------

•If has some error when insert data in the database, 
verify is is all correct and run the code: npx knex
migrations:latest and restart the nodemon.

------------------------------------------------------

•Use all methods (functionalities) of an entity in an
 even controller file.

------------------------------------------------------

•Use the command mkdir name_of_folder to create a new 
folder by the cmd prompt.

------------------------------------------------------

•You can to custom your npm commands changing the name 
of the script linked with the command inside the pack-
age.json in the property scripts.

------------------------------------------------------

•The browsers always use the method GET to show web 
pages.

------------------------------------------------------

•Always when using a response method, use the return 
prefix before to force the code stop at this point.

------------------------------------------------------

•Create a new workspace in the Inmsomia always start a 
new project.

------------------------------------------------------

•Be careful to use the import command, this can be tri-
gger unexpected errors by the autocomplete of the Visu-
al Studio Code.

------------------------------------------------------

•While works with typescript files, all files also need 
be in .ts format, using react/react native the format is
.tsx.

------------------------------------------------------

•The order that the files are inside the folder migrations 
is the same order with the files will to run in the knex 
command. Be careful when sorting your files. Example: 
01_create_users.ts and 02_change_user.ts.

------------------------------------------------------

•Always import with the first letter in Capital case a 
package when using typescript.

------------------------------------------------------

•The native variable __dirname always return the di-
rectory of the current file.
