=======================DOM OBJECT==========================

--------------------------PROPS----------------------------

activeElemnt:
Returns the currently focused element in the document.

anchors:
Returns a collection of all <a> elements in the document
that have a name attribute.

body:
Returns the <body> element.

cookie:
Returns all name/value pairs of cookies in the document.

characterSet:
Returns the character encoding for the document.

defaultView:
Returns the window object associated with a document, or 
null if none is available.

designMode:
Controls whether the entire document should be editable.

documentURI:
Sets or returns the location of the document.

domain:
Returns the domain name of the server thar loaded the docu-
ment.

embeds:
Returns a collection of all <embed> elments in the docuemnt.

forms:
Returns a collection of all <form> elements in the document.

fullscreenElement:
Returns the current element that is diplayed in fullscreen 
mode.

head:
Reurtns the <head> element of the document.

images:
Return the colletion of all <img> elements in the document.

lastModified:
Returns the date and time the docuemtn was last modified.

links:
Returns a collection of all <a> an <area> elements in the 
document that have a href attribute.

readyState:
Returns the loading status of the document.

referrer:
Returns the URL of the document that loaded the current docu-
ment.

scripts:
Returns a collection of <script> element in the document.

title:
Sets or returns the title of the document.

URL:
Returns the entire URL od the HTML document.


--------------------------METHODS----------------------------

addEventListener():
Attaches an event handler o the document.

adoptNode():
Adopts a node from another document.

close():
Closes the output strem previously opened with document.open().

createAttribute():
Creates an attribute node.

createComment():
Creates a comment node with specified text.

createDocumentFragment():
Creates an empty DocumentFragment node.

createElement():
Creates an Element node.

createEvent():
Creates a new event.

createTextNode():
Creates a Text node.

fullscreenEnabled():
Returns a boolean value idicating whether the document can be 
viewed in fullscreen mode.

getElementById():
Returns the elment that has the ID attribute with the name spe-
cified.

getElementsByClassName():
Returns a HTML collecion containing all elements with the speci-
fied value.

getElementsByTagName():
Returns a HTML collection containing all elements with the spe-
cified tag name.

hasFocus():
Return a boolean value indicating if whether the document has 
focus.

importNode():
Imports anode from another document.

normalize():
Removes empty Text nodes, and joins adjancent nodes.


open():
Opens a HTML output strem to collect output from document.write().

querySelector():
search for an element in a exact position in the document ex: 
querySelector('div#container input') gets the input inside 
the div with id container or querySelector('buttton[name=calc]').

querySelectorAll():
Returns a statice NodeList containing all elements that matchse a 
specified CSS selector(s) in the document.

removeEventListener():
Removes an event handler addded bu the addEventlisterner() method 
from the document.

renameNode():
Renames the specified node.

wirte():
Writes HTML expression or JavaScript code to the document.

writeIn():
Writes HTML expression or JavaScript code to the  document adding 
a newline character after each statement.
