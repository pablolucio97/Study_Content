
=========DOING DEPLOY OF REACT APP USING NETLIFY=========

1) install the netlify globally if do not has made it:
npm install netlify-cli -g

---------------------------------------------------------

2) Navigates to your directtory project and run the 
command npm run build (this will create a folder build)

---------------------------------------------------------

3)Log with netlify running: netlify login

---------------------------------------------------------

4)To create yuor react deploy, run: netlify deploy and 
choice a new site.

---------------------------------------------------------

5)When facing with 'deploy patch' type 'build' and await
the automatic deploy.
