====================REACT NATIVE BUNDLE VISUALIZER=========================

The React Native Bundle Visualizer is a lib that allow you to see graphi-
cally through boxes, the size of each pack in your app.

1 Run to install yarn add react-native-bundle-visualizer --expo managed the 
React Native Bundle Visualizer if you did created your app with Expo Mana-
ged or run yarn add react-native-bundle-visualizer -D if you did created 
with Expo bare workflow.

2) Run yarn run react-native-bundle-visualizer to start the React Native 
Bundle Visualizer.

3) The React Native Bundle Visualizer will open in a new tab on browser.

Obs: If you can't open the React Native Bundle Visualizer, install the sour-
map explorer running: yarn add global source-map-explorer
