
---------------------------CREATING SEEDS------------------------------

Seeds are useful to populate database with default values.

1)Add the script "knex:seed" to your pakcage.json:
"knex:seed" : "knex --knexfile knexfile.ts seed:run"

2)Configure your kenxfile to generate seeds:

seeds:{
    directory: path.resolve(__dirname, 'src', 'database', 'seeds')
},

3)Inside of the folder 'src', create a new folder named 'seeds' and a 
new file named default_seedsname and create your seed. 

Example (In this example will be added three objects to the user with id 
= 1 inside the table 'projects'):

import Knex from 'knex'

export async function seed(knex : Knex){
    await knex('projects').insert([
        {title : 'React', user_id : 1},
        {title : 'React Native', user_id : 1},
        {title : 'Node JS', user_id : 1}
    ])
}

4)Run the script "knex:seed" (nom run knex:seed) to see your seeds inside 
the table projects.

OBS: To generate seed, the field sented by the seed should exists in the 
destiny table.
