
=============CREATING A NEW NODE JS PROJECT=================

1) Create a new folder to the project and in the folder 
project starts a new node project with npm init -y (it 
will create a new package.json file). 

2)Install the express with the command npm install express  
and install the nodemon like dev dependency using npm ins-
tall nodemon -D.

3)Inside the file package.json create a new script to start 
the nodemon with the content 'nodemon server.js'. Example:
"dev": "nodemon server.js".

4)Execute the command npm run dev to starts the server.
