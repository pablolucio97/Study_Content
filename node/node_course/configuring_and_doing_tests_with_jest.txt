
================Configuring and doing tests with Jest=================

1)Install the Jest: npm install jest -d

2)Run the code npx jest init.

3)Create a folder named tests and inside this, another folder named 
integration.

4)Create a file to your test with the extension spec.js


=========================Doing single test============================

describe('Description of your test', () => {
it('Description of the result expected', () =>{
expect(2+2).toBe(4)
})
})
