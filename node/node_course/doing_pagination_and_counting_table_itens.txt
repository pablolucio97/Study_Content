------------------DOING PAGINATION----------------------

Paginations always be made in the index method.

Example: Doing a pagination to show 5 itens for page:

const {page = 1} = request.query
const filteredProject =  db('projects')
.limit(5).offset((Number(page) - 1) * 5)

Route query: http://localhost:3334/list-projects?page=3
